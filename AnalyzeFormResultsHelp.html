Remove Duplicate Responses

Removing duplicate responses based on spcific item.
For example, if the first item in the form is an email address, this method will remove any other response with the same email address.
Responses with the same email address on other form will NOT removed.

Remove Unauthorized Responses

Removing responses from a form based on approved keys. Like the removing duplicates option, this function is based on the first question in the form.
For example, Lets say the first question is an email address and you want to delete any response from witch not exist in specific list.
For this method, you'll need to add all the email addresses the column A on sheet 'Approved List' (see the following example). Responses with the same email address on other form will NOT removed.

  • All questions up to your primary question must be 'requiered'. Normally, it's the first question.

    • How to Use

  • On the spreadsheet, select the range where the forms URLs.


  • Select primary question number.


  • Press 'Remove Duplicate Responses / Remove Unauthorized Responses' and wait.
